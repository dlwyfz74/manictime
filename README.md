## 它究竟在记录什么？

ManicTime 是由斯洛文尼亚开发商 Finkit d.o.o. 出品的 Windows 时间追踪工具，软件装好之后基本不用管它，它自己在后台默默收集数据：

- 你用了哪些程序，各用了多久

- 你什么时候离开电脑、离开了多久
- 你的屏幕截图（专业版），留存每个时刻的工作状态

这些数据全部存在本地，不上传、不外发。你不联网它也能用，隐私方面不需要担心。
